增值业务查询

返回

qryProductInfo

获取用户已办理的增值业务数据。

支持格式

JSON / XML

URL

     XML：http://cservice.client.189.cn:8004/map/clientXML
     JSON：http://cservice.client.189.cn:8004/map/clientJSON

HTTP请求方式

POST

是否需要登录

是
关于登录授权，参见 如何登录授权

请求参数

字段 必选类型及范围说明
PhoneNbr
1string需要查询的用户手机号码。

  <Content>
    <Attach>value</Attach>

  </Content>
