请求参数

字段 必选类型及范围说明
OptPackages
1Object可选包信息结构体
ShopId1string店铺ID
GpFlag1int团购秒杀标识，取值：
0：普通订单
1：团购订单
2：秒杀订单
ChildType?string子类型
用于“靓号”相关接口时不可省略。
取值：
881     云卡
882     积木
883     纯流量
884     乐享上网
885     乐享聊天
886     19元包年打
UimCode?
string
UIM卡编码

UserId1
string
用户ID
NewPhoneNumber1
bool
是否新号码
false：旧号码
true：新号码
PhoneNumber1
string
手机号码
ComboId1
string
套餐ID
CashSalesProdId1
string
现金预存包ID
该字段内容为空字符串时，代表没有现金预存包
SalesProdId1
string
销售品ID
BlockInfo
1Object积木套餐信息
ContractId?
string
合约ID
SubPhoneNumber1
string
销售品关联号码（副号）
无副号时，设置为空字符串
DsitChannel?
string
分销渠道，客户端不关心该字段。
格式：
  [src].[channel].[cid].[wi]
例：
  189.cps.XXXXXXXXXXXX.abc

OptPackages：
字段 必现类型说明
OptProdId
1List可能包含多个可选包包Id
OptSalesProdId1string可选包销售品ID

BlockInfo：
字段 必选类型及范围说明
OptCodes
1List可选包产品NBR编码
SmsCode1string短信包产品NBR编码（未选择时为空字符串）
FlowCode1
string
流量包产品NBR编码（未选择时为空字符串）
VoiceCode1
string
语音包产品NBR编码
