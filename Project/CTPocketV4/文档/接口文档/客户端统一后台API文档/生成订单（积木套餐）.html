生成订单（积木套餐）

返回

JuggleGgroup

生成一个积木套餐的订单。

支持格式

JSON / XML

URL

     XML：http://cservice.client.189.cn:8004/map/clientXML
     JSON：http://cservice.client.189.cn:8004/map/clientJSON

HTTP请求方式

POST

是否需要登录

是
关于登录授权，参见 如何登录授权

请求参数

字段 必选类型及范围说明
ShopId
1string店铺编码
UserId1
string
用户ID
SalesProdId1
string
积木套餐、积木套餐+终端或积木套餐+云卡销售品ID.
BlockInfo1Object积木套餐信息对象
PhoneNumber1
string
手机号码
NewPhoneNumber1
string
是否新号码
false：旧号码
true：新号码
GpFlag1
string
团购秒杀标识，取值：
0：普通订单
1：团购订单或秒杀订单

BlockInfo：
字段 必选类型及范围说明
OptCodes
?List可选包产品NBR编码
SmsCode1string短信包产品NBR编码（未选择时为空字符串）
FlowCode1
string
流量包产品NBR编码（未选择时为空字符串）
VoiceCode1
string
语音包产品NBR编码
