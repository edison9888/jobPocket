筛选百搭销售品

返回

filterSlsPrdList

新百搭销售品购买功能使用，用于筛选销售品信息。

支持格式

JSON / XML

URL

     XML：http://cservice.client.189.cn:8004/map/clientXML
     JSON：http://cservice.client.189.cn:8004/map/clientJSON

HTTP请求方式

POST

是否需要登录

是
关于登录授权，参见 如何登录授权

请求参数

字段 必选类型及范围说明
ShopId
1string店铺ID
Spec？string筛选的规格key （4.2暂不使用）
Value？string筛选的规格value （4.2暂不使用）
Sortby？string排序条件：
1 按照销量升序
2 按照销量降序
3 按照价格升序
4 按照价格降序
5 按照套餐金额升序
6 按照套餐金额降序
Type?int销售品类型
Index?int页码
PageSize?int每页数量
KeyWord？string搜索关键字（需使用中文编码，参考附录）
