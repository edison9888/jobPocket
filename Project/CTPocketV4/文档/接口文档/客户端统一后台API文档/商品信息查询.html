商品信息查询

返回

igProdList

查询商品信息。

支持格式

JSON / XML

URL

     XML：http://cservice.client.189.cn:8004/map/clientXML
     JSON：http://cservice.client.189.cn:8004/map/clientJSON

HTTP请求方式

POST

是否需要登录

是
关于登录授权，参见 如何登录授权

请求参数

字段 必选类型及范围说明
PageIndex
1int页码
PageSize1int每页显示数量
Sort1int排序规则：
1:单价从低到高(默认) ；2单价从高到低；3：销量从低到搞；4：销量从高到低
MinPrice1int最低价格
MaxPrice1int最高价格
CommdityId？string商品ID，可不传，可传多个用“｜”分开
KeyWord？string关键字（商品类别、商品品牌、商品名称搜索）
CategoryId？string商品大类ID（虚拟礼品:08）
SmallCategoryId？string商品小类ID（
电信产品：0816
电子卡券：0806
店面兑换：0817
）
