Imsi反查认证

返回

imsiInfo

通过IMSI信息反查方式获得登录用户信息。

支持格式

JSON / XML

URL

     XML：http://cservice.client.189.cn:8004/map/clientXML
     JSON：http://cservice.client.189.cn:8004/map/clientJSON

HTTP请求方式

POST

是否需要登录

是
关于登录授权，参见 如何登录授权

请求参数

字段 必选类型及范围说明
ImsiNbr
1string需要查询的用户IMSI码。

XML请求示例

<Request>
  <HeaderInfos>
    <ClientType>3.0.2</ClientType>
    <Source>100001</Source>
    <SourcePassword>F56c</SourcePassword>
    <Token></Token>
    <UserLoginName>test</UserLoginName>
    <Code>imsiInfo</Code>
    <Timestamp>20130511084030</Timestamp>
  </HeaderInfos>
  <Content>
    <Attach>test</Attach>
    <FieldData>
      <ImsiNbr>460004722458</ImsiNbr>
    </FieldData>
  </Content>
</Request>
