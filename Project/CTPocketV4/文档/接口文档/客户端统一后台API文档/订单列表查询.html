订单列表查询

返回

qryOrderList

获取订单列表信息。

支持格式

JSON / XML

URL

     XML：http://cservice.client.189.cn:8004/map/clientXML
     JSON：http://cservice.client.189.cn:8004/map/clientJSON

HTTP请求方式

POST

是否需要登录

是
关于登录授权，参见 如何登录授权

请求参数

字段 必选类型及范围说明
UserId
1string用户ID
PageIndex1string页码
PageSize1
string
每页显示数量
Type？string订单状态类型，如果不传默认全部。
0：全部
1：有效订单
2：已取消订单
3：待支付订单
4：待收货订单
5：已完成订单
StartTime?
string
查询开始时间（YYYY-MM-DD）
EndTime?
string
查询结束时间（YYYY-MM-DD）

返回字段说明


字段 必现类型说明
TotalCount
1string总条目数
DataList1Object订单内容对象（可能有多个）

DataList：
字段 必现类型说明
AbType
？string订单AB类型
取值如下：
1：A1
2：A2
3：B
Price1string价格
OrderPayedDate1
string
订单支付时间
PayConfirmedTime？
string
支付确认时间（仅供支付确认功能使用）
ExchangeStatus1
string
是否能够退换货
0x00:不能退货、不能换货
0x10:能够退货、能够换货
0x20:能够换货
ConsigneeName1
string
收货人
OrderType1
string
订单类型
PayConfirmed？
string
是否支付确认（仅供支付确认功能使用）
OrderCreatedDate1
string
订单生成时间
OrderStatusDescription1
string
订单状态文字描述
OrderStatusCode1
string
订单状态编码
ShopId1
string
店铺编码
历史订单该编码为固定值：99999
Gifts1Object整单关联赠品信息（只包含整单赠品）
OrderId1
string
订单号
Items1Object订单商品明细（可能包含多个）

Items：
字段 必现类型说明
SalesProName
1string销售品名称
ComboDescription1
string
套餐描述
Count1
string
该销售品总数量
Remark1
string
订单备注
AreaCode1
string
销售品所属市编码
PhoneNumberPrestoreAmount1
string
销售品关联号码预存款金额
ProvinceCode1
string
销售品所属省编码
ExchangeStatusDescription1
string
换货状态描述
（仅用于“获取可换货与换货状态订单列表”接口，其他接口应忽略该字段）
SalesProId1
string
销售品ID
Commented1
string
购买用户是否已评论
NewPhoneNumber1
string
合约机号码为新购号码
SalesProdType1
string
销售品类型
ExchangeStatusCode1
string
换货状态编码

销售品未处于换货状态时，该字段值为空字符串（长度为0的字符串）
（仅用于“获取可换货与换货状态订单列表”接口，其他接口应忽略该字段）
Price1
string
销售品单价(原)
PhoneNumber1
string
销售品关联号码
无关联号码时，设置为空字符串
PhoneNumberPrice1
string
销售品关联号码价格
Gifts1Object销售品关联赠品数组（暂时忽略）
ContractDescription1
string
合约描述
RealPrice1
string
销售品单价（最终）
SalesProIconUrl1
string
销售品图片URL

其他

无
