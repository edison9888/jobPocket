返回字段说明


字段 必现类型说明
AccAmountFlow1string流量总量【为上面数组中流量的累加值】(暂无意义)
UsedAmountFlow1
string
已使用流量【为上面数组中流量的累加值】(暂无意义)
UnitTypeIdFlow1
string
流量的单位类型，固定为3（KB）(暂无意义)
SmsAccList1List短信累积量包含1个ListItem
CallAccList1
List
通话时长累积量包含1个ListItem
FlowAccList1
List
流量累积量包含1个ListItem
PpaccuAccList1
List
套餐情况包含多个ListItem

ListItem：
字段 必现类型说明
ProdOfferName
1string套餐名称
AccList1List套餐信息结构体(可能有多个Item)

Item：
字段 必现类型说明
AccAmount
1string累积量总量
AccuRscName1string累积量名称
BalanceAmount1string累积量剩余量
UnitTypeId1string单位类型:0：分（金额） 1：分钟（时长）2：次数 3：流量（KB）
EndTime1string累计周期结束时间

其他

无
