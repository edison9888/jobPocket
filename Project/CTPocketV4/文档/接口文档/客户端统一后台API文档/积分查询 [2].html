积分查询

返回

igInfo

查询用户积分信息。

支持格式

JSON / XML

URL

     XML：http://cservice.client.189.cn:8004/map/clientXML
     JSON：http://cservice.client.189.cn:8004/map/clientJSON

HTTP请求方式

POST

是否需要登录

是
关于登录授权，参见 如何登录授权

请求参数

字段 必选类型及范围说明
DeviceNo
1string手机号码
DeviceType1string设备类型（手机用户目前固定取值为7）
ProvinceId1string省份ID（掌厅用户默认取值为35：全国中心）
CustId1
string
用户ID
QueryType1
string
查询类型：
0：积分+抵用券都要查询过期时间；
1：积分+抵用券不显示过期时间；
2：只查积分并显示多少积分年底过期；
3：只查积分，不显示年底过期积分；
4：只查抵用券
