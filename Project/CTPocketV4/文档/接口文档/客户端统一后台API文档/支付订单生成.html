支付订单生成

返回

payOrder

根据订单生成支付信息，用于支付插件发起支付请求。

支持格式

JSON / XML

URL

     XML：http://cservice.client.189.cn:8004/map/clientXML
     JSON：http://cservice.client.189.cn:8004/map/clientJSON

HTTP请求方式

POST

是否需要登录

是
关于登录授权，参见 如何登录授权

请求参数

字段 必选类型及范围说明
PhoneNumber
1string用户手机号码。
OrderId1string需要发起支付的订单ID
