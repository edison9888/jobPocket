流量提醒

返回

qryFlowRemindStateInfo

查询推送流量提醒状态信息。

支持格式

JSON / XML

URL

     XML：http://cservice.client.189.cn:8004/map/clientXML
     JSON：http://cservice.client.189.cn:8004/map/clientJSON

HTTP请求方式

POST

是否需要登录

是
关于登录授权，参见 如何登录授权

请求参数

字段 必选类型及范围说明
PhoneNum
1string用户手机号
TimeStamp1
string
最后一次成功收到发起提醒反馈，并完成提醒的时间。格式： YYYY-MM-DD
默认传0
