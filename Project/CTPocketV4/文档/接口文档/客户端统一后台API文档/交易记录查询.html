交易记录查询

返回

igOrderList

获取积分交易记录信息。

支持格式

JSON / XML

URL

     XML：http://cservice.client.189.cn:8004/map/clientXML
     JSON：http://cservice.client.189.cn:8004/map/clientJSON

HTTP请求方式

POST

是否需要登录

是
关于登录授权，参见 如何登录授权

请求参数

字段 必选类型及范围说明
PageIndex
1string页码
PageSize1
string
每页显示数量
DeviceNo1
string
手机号
DeviceType1
string
设备类型（手机用户目前固定取值为7）
ProvinceId1
string
省份ID（掌厅用户默认取值为35：全国中心）
CustId1
string
用户ID
OrderId？
string
订单ID，如果不传则查询多条记录。
Status1
string
订单状态：0未支付，1已支付，-1已撤销，-9返销 ，-2全部
DataType1
string
订单创建时间：
0： 全部，1一个月内，2三个月内，3三个月以前的（六个月内）
